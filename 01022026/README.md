Пример  управления  шаговыми  двигателями NEMA 17 через ESP32 на прямую.

В примере описано как управлять синхронно 3 шаговыми  двигателя, деление  шага  1.8 градуса или 200 шагов  на оборот.

В примере  есть управление тестовое и вывод информации  об оборотах на дисплей по i2c шине.

При меры  команд:

X100; X250; X-240; X-10; Сделает 1.25 круга и вернется назад в изночальное положение.

Y100; Y500; Y-600;

Z400; Z400; Z-500; Z-300;

Для  работы  используются  драйвера  А4988. Для асинхронной  работы  рекомендуется использовать  библиотеки.
В примере реализован лишь пример.
