В примере приведено управление на основе библиотеки, когда сразу  три  двигателя шаговых могут работать асинхронно.

Каждому двигателю через команду можно  задать позицию к которой он должен вращаться, примеры команд довольно простые.

Пример команд реазиванные в примере:

X1000; X500; Y400; Y-2000; Z500; Z2000; X100; X-900;

Все команды  идут через сериал порт. У библиотеки есть зависимости не  забудьте их поставить.
